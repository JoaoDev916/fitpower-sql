# FitPower Academia — Projeto SQL

Projeto com scripts SQL para gerenciamento da academia FitPower:
- create_tables.sql (criação das tabelas)
- insert_data.sql (povoamento inicial)
- select_queries.sql (consultas com JOIN, WHERE, ORDER BY, LIMIT)
- update_delete.sql (3 UPDATEs e 3 DELETEs)

## Como executar
1. Criar banco (ex.: fitpower_db) no MySQL/Postgres.
2. Executar create_tables.sql.
3. Executar insert_data.sql.
4. Rodar as consultas em select_queries.sql.
5. Testar update_delete.sql conforme necessidade.
